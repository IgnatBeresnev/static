createDispatcher

open override fun createDispatcher(allFactories: List<MainDispatcherFactory>): MainCoroutineDispatcher(source)

Creates the main dispatcher. allFactories parameter contains all factories found by service loader. This method is not guaranteed to be idempotent.

It is required that this method fails with an exception instead of returning an instance that doesn't work correctly as a Delay. The reason for this is that, on the JVM, DefaultDelay will use Dispatchers.Main for most delays by default if this method returns an instance without throwing.
