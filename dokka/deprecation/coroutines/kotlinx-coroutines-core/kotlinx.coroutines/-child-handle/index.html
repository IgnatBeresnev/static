abstract fun childCancelled(cause: Throwable): Boolean

Child is cancelling its parent by invoking this method. This method is invoked by the child twice. The first time child report its root cause as soon as possible, so that all its siblings and the parent can start cancelling their work asap. The second time child invokes this method when it had aggregated and determined its final cancellation cause.
