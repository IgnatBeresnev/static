abstract fun tryResume(value: T, idempotent: Any? = null): Any?(source)

Tries to resume this continuation with the specified value and returns a non-null object token if successful, or null otherwise (it was already resumed or cancelled). When a non-null object is returned, completeResume must be invoked with it.

When idempotent is not null, this function performs an idempotent operation, so that further invocations with the same non-null reference produce the same result.

This is unstable API and it is subject to change.**
