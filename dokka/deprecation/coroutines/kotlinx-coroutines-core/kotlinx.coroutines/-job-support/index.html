A concrete implementation of Job. It is optionally a child to a parent job.

This is an open class designed for extension by more specific classes that might augment the state and mare store addition state information for completed jobs, like their result values.

Returns true if this job was cancelled for any reason, either by explicit invocation of cancel or because it had failed or its child or parent was cancelled.
